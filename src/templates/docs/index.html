Bienvenido

Esta es la documentacion de APISII. Aquí encontrarás lo que necesitas para emplementar de manera rápida esta API en tu sistema. Cualquier duda o sugerencia, o si encuentras algun error, no dudes en contactarnos.

API Keys

Nuestro sistema utiliza API Keys para autenticar que la API está siendo utilizada por un usuario registrado. Para esto, cada usuario tiene su API Key individual, la se puede obtener haciendo click en el botón "Copiar API Key Personal", la cual sirve para autenticar al usuario en las llamadas de usuario de la API. Además, cada empresa tiene su propia API Key, la cual se obtiene haciendo click en el boton "Copiar API Key" correspondiente a la empresa, que sirve para autenticar llamadas exclusivamente a la empresa deseada, sin que se pueda acceder a las otras empresas registradas.

Llamadas de usuario

Las llamadas de usuario permiten obtener los datos que las llamadas de empresa, pero en el caso de los usuarios, se pueden ver todas las empresas registradas utilizando la misma API Key.

Resumen de Compras

  • apiKey: corresponde a tu apiKey de usuario, disponible en tu perfil.
  • R.U.T.: el RUT de alguna de las empresas registradas en tu perfil.
  • Contraseña: contraseña del Servicio de Impuestos Internos (SII) de la empresa correspondiente.
  • Periodo: periodo tributario que se desea consultar en formato Año-Mes (Ej: Julio de 2022 corresponde a 202207).
